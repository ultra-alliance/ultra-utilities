Ultra Utilities

ultra-utilities is a collection of tools made by the Ultra Tech Alliance to help developers build on the Ultra blockchain. Before anything else, you should read the official Ultra documentation.

Packages

Note: All packages are still in development and are not ready for production A lot of things may change, including packages names, APIs, etc.

ultra-sdk

JS/TS SDK to interact with the Ultra blockchain.

react-ultra

A react wrapper of the ultra-sdk to interact with the UOS.

uikit

UI Kit based on a Material UI theme.
