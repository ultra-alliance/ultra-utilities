Description

This module includes various enumerations, classes, and type aliases that facilitate the communication with the blockchain, such as authentication, tokenization, and smart contract interaction. It also provides a set of HTTP functions, formatters, and other utilities that streamline the development of UOS-connected applications.

‚öôÔ∏è Quick Start

You can install the package by running the following command:

npm install @ultra-alliance/ultra-sdk

Then you can import functions, and classes from the SDK in your project:

import { getBlock, Ultra } from '@ultra-alliance/ultra-sdk';

üöÄ Usage

If all methods are exported by the package, we also provide a Ultra class that allows you to instantiate a new Ultra instance with a given bpApiEndpoint (default will be DEFAULT_BP_API_ENDPOINT). This class is a wrapper of the SDK functions.

import { Ultra } from '@ultra-alliance/ultra-sdk';

const ultra = new Ultra({
 bpApiEndpoint: 'https://api.ultra.eossweden.org',
});

ultra.getAccount("aaa1bbb2ccc3").then((account) => {
  console.log(account);
});

This SDK provides a set of functions, formatters and utils that allow you to interact with the blockchain, among others:

APIs
  • getUosBalance to get the UOS balance of an account
  • getBlock to get a block by its number
  • getAccount to get an account-detail by its name
  • getAbi to get the ABI of a smart contract
  • getTableRows to get the rows of a table
  • getUniqDetail to get the details of a uniq
  • getUniqsOwned to get the uniqs owned by an account
  • getInfo to get the blockchain info
  • getListedUniqs to get the uniqs listed on the marketplace

Utilities
  • getZipContent to get the content of a zip file and read the Uniq manifest

Calculations
  • calcTotalPrice to calculate the price value of a currency for a given amount in a given base price.

Formatters
  • formatUosBalance to format a UOS balance
  • formatName to format a name (eg: aa1...cc3)
  • formatCurrencyValue to format a currency value
